# use-podman

## Installation

```bash
cd ~/.claude/skills
git clone https://github.com/jarrettmeyer/use-podman.git
```

### Updating

After the initial installation, update this skill by pull changes.

```bash
cd ~/.claude/skills/use-podman
git pull origin main
```

## Rationale

Some organizations may prefer `podman` over `docker` due to commercial licensing differences.
